## 第一阶段：Python + AI 赋能基础筑基期（第1~25天）

**目标**：建立编程思维，掌握 Python 核心库，熟练使用 AI 协作工具。

> ⚠️ **阶段考核**：T21~T25 进行结业考核，通过后方可进入第二阶段。

| 周次 | Task 范围 | 核心内容 |
|------|-----------|---------|
| 第1周 | T01~T05 | 编程认知、环境搭建、AI协作入门、Python基础语法 |
| 第2周 | T06~T10 | NumPy/Pandas 数据处理、Matplotlib/Seaborn 数据可视化 |
| 第3周 | T11~T15 | EDA 实战、AI 辅助优化代码、Python工程规范 |
| 第4周 | T16~T20 | 闯关项目、综合练习 |
| 第5周 | T21~T25 | **结业考核**（必过才能进入第二阶段） |

### 每周详细安排

#### 📌 第1周：编程认知与基础（T01~T05）

- 编程思维建立，理解计算机如何"思考"
- Python 开发环境搭建（IDE / 包管理 / 虚拟环境）
- AI 编程助手入门：如何用 AI 加速学习而非替代思考
- 基础语法：变量、数据类型、控制流（if/for/while）、函数
- 小练习：用 Python + AI 完成第一个小型脚本

#### 📌 第2周：数据处理与可视化（T06~T10）

- **NumPy**：数组操作、矩阵运算、统计函数
- **Pandas**：DataFrame 读写、筛选、分组、聚合
- **Matplotlib**：折线图、柱状图、散点图基础绘图
- **Seaborn**：统计图表、热力图、分布图
- 练习：对真实数据集做基本探索性分析

#### 📌 第3周：EDA 实战与工程规范（T11~T15）

- 完整 EDA 流程：数据概览 → 缺失值处理 → 异常值检测 → 特征分析
- AI 优化代码：让 AI 帮你 review 和改进代码
- 工程规范：项目结构、命名规范、注释习惯、Git 基础
- 实践：对一个中型数据集独立完成 EDA 报告

#### 📌 第4周：闯关项目与综合练习（T16~T20）

- 综合性闯关任务，串联前三周所学
- 涵盖数据采集 → 清洗 → 分析 → 可视化全流程
- 组内互评，互相学习编码风格和分析思路

#### 📌 第5周：结业考核（T21~T25）

- 综合考核：限时完成一个数据分析任务
- 考核内容包括：数据获取、Pandas 处理、可视化呈现、结论撰写
- **通过考核才能进入第二阶段实战**

## 第二阶段：全真项目实战落地期（第26~45天）

**目标**：独立完成一个数据分析 / 可视化项目，产出可展示的作品。

| 周次 | Task 范围 | 核心内容 |
|------|-----------|---------|
| 第6周 | T26~T30 | 项目选题、数据清洗、多维度分析 |
| 第7周 | T31~T35 | 相关性分析、商业假设验证 |
| 第8周 | T36~T40 | 报告撰写、图表优化、互评迭代 |
| 第9周 | T41~T45 | 成果归档、结业评优、证书发放 |

### 每周详细安排

#### 📌 第6周：选题与数据准备（T26~T30）

- **项目选题**：从多个真实数据集中选择感兴趣的方向
- **数据清洗**：处理缺失值、重复值、格式统一
- **多维度分析**：从时间、地域、类别等维度切入
- 输出：数据清洗报告 + 初步分析笔记

#### 📌 第7周：深度分析与假设验证（T31~T35）

- **相关性分析**：变量间关系探索，计算相关系数
- **商业假设验证**：提出假设 → 数据验证 → 得出结论
- 输出：分析文档 + 关键可视化图表

#### 📌 第8周：报告撰写与迭代（T36~T40）

- **报告撰写**：将分析过程组织为结构化报告（背景 → 方法 → 发现 → 结论）
- **图表优化**：配色、排版、故事线打磨
- **互评迭代**：组内交叉评审，根据反馈修改完善
- 输出：完整数据分析报告初稿

#### 📌 第9周：成果归档与结业（T41~T45）

- **成果归档**：项目代码 + 数据 + 报告整理打包
- **结业评优**：评选优秀项目进行展示
- **证书发放**：完成全部学习任务的学员获得结业证书
