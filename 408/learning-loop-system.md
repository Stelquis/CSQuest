## 入门指南：从哪一章开始学？

### 四科推荐入门顺序

| 梯队 | 科目与章节 | 理由 |
|:-----|:-----------|:------|
| **第一梯队（先学）** | 数据结构：线性表 → 栈/队列 → 树 → 图 | 最基础的逻辑结构，后续三科都用到 |
| **第一梯队（先学）** | 计组：数据表示 → 运算器 → 指令系统 | 理解计算机的"语言"和基本运算 |
| **第二梯队（需要前置）** | OS：进程管理 → 内存管理 | 需要计组-指令系统作为前置 |
| **第二梯队（需要前置）** | 计网：网络层 → 传输层 | 体系感强，不需要太多前置 |

### 跨科缝合点标注（提前知晓，学到时注意）

```
Cache（计组）↔ 页表（OS）
→ 学完 Cache 后，等 OS 学到内存管理时再缝合

指令流水线（计组）↔ 进程上下文切换（OS）
→ 学完两者后，单独找一道联考题做缝合测试
```

### 避坑注意

- ❌ 不要从"计组-浮点数运算"开始（前置要求高，可能卡死）
- ❌ 不要从"计网-物理层"开始（细节多且低频，容易劝退）
- ✅ 数据结构前三章是最安全的起点

## 第一章：学习的元流程（拿到任何一章，都走这 5 步）

如果你不知道该从哪下手，就严格执行这 5 个步骤，不要跳步。

### Step 1：定位——这章在"计算机"里解决什么问题？

打开教材前，花 3 分钟做这件事：

- 翻开目录，看这章的标题
- 问自己：**"如果没有这个技术，计算机会出什么 bug？"**

> 举个例子：学"Cache（高速缓存）"前，问自己："如果 CPU 每次都去内存取数据，会发生什么？"（答：CPU 太快，内存太慢，CPU 大部分时间在"等"，效率极低。）→ Cache 就是为了解决"速度不匹配"这个矛盾。

**为什么要做这一步？**
- 带着问题读，你是"主动侦察兵"
- 不带着问题读，你是"被动垃圾桶"

### Step 2：定级——这章值不值得花时间？

打开王道，看这章的**历年真题考察频率**。

| 级别 | 特征 | 你的策略 |
|:-----|:-----|:---------|
| **🏗️ 梁柱（高频）** | 每年都考，常出大题 | **砸时间**：完整闭环（图谱 + 做题 + 深度纠错） |
| **🧱 砖墙（中频）** | 隔年考，多为选择题 | **正常学**：画简易图谱 + 做选择题 + 轻度纠错 |
| **🎨 涂料（低频）** | 偶尔考 1-2 分 | **混眼熟**：教材划线 + 看答案，不进错题本 |

**教材怎么判断考频？** 王道每章开头都有"考情分析"，直接看那个就行。

### Step 3：预习——带着 Step 1 的问题，快速扫一遍

- 时间：**15 分钟**
- 内容：只读**标题、图表、加粗定义**
- 目标：搞清楚 Step 1 的问题大概被"怎么解决的"
- 产出：在草稿纸上写下 3-5 个这章的关键词（后面要用的）

> 这一步是很多人忽略的。直接一头扎进细节，越学越乱。

### Step 4：精读——三层加工法

**注意：只对"梁柱"级执行完整三层，"砖墙"只做 L1，"涂料"直接跳过这一步。**

#### L1 结构化——画关系图

- 工具：一张白纸
- 任务：把这一章的核心概念**按照逻辑关系**画出来
- 判断标准：画完之后，你能**指着图**说出"A 和 B 是什么关系"

#### L2 关联化——找"隔壁邻居"

- 任务：回答一个问题——"这章的知识，在 408 其他三科里对应什么？"
- 产出：在笔记旁边写一句 `⛓️ → [科目]-[概念]`
- 例子：学 OS 的"页表" → 写 `⛓️ → 计组-Cache-组相联映射`
- 判断标准：你能说清"数据从这科流到那科，中间经过了什么"

#### L3 模型化——抽象成一句话

- 任务：把这个知识点**压缩成 1 个思维模型**
- 常见模型：状态机、资源分配、缓存层叠、流水线
- 例子：TCP 拥塞控制 → 状态机；PV 操作 → 资源分配与互斥
- 判断标准：你能用这个模型**解释 2 道以上真题**

**L3 如果写不出来，不勉强。它是在做题中慢慢长出来的。**

### Step 5：费曼检验——讲给小白听

- 找一个不考研的同学，或者对着空气讲
- 要求：**不用专业术语**，用大白话把这一章的核心讲清楚
- 卡壳的地方 → 那就是你没搞懂的地方 → 回 Step 4 补

> **费曼检验的标准：如果你在解释中用了"显然""众所周知"，说明你自己也没真懂。**

### 数据源清洗（用什么材料学）

不是所有材料都值得花时间。只有**高质量输入**才配得上你的加工：

```
教材金线（只这三本）：
├── 《王道考研》四本单科书 —— 主力，覆盖 90% 考纲
├── 袁春风《计组》—— 查漏补缺用，前命题人
└── 谢希仁《计网》—— 查漏补缺用

视频金线：
├── 首选 B 站"3 分钟"动画讲解 —— 用最短路径理解核心矛盾
├── 超过 10 分钟的扩展视频 → 划入涂料级，不看
└── 王道基础课视频 —— 跟着学完每章的基础概念
```

## 第二章：做题方法论（正确率是检验的唯一标准）

### 先问 3 个问题再动笔

拿到一道题，先不要急着算。问自己：

1. **这道题在考什么概念？**（是 L1 结构化里的哪个节点？）
2. **命题人设了什么陷阱？**（选项里哪两个长得像？）
3. **如果是大题，得分点在哪里？**（老师按什么步骤给分？）

### 选择题——三步走

1. **排除**：先划掉两个明显错的
2. **定位**：剩下两个，辨析它们对应的概念区别
3. **极端检验**：选项里有"一定/永远/所有" → 大概率是错的

### 大题——四步走

1. **拆题**：题干给了什么条件？要求算什么？
2. **匹配范式**：这道题属于哪种模型？（状态机？资源分配？）
3. **执行**：按步骤写，每一步都要写依据
4. **验证**：代入极端值检查结果是否合理

### 算法题——能跑通最重要

1. 先写暴力解 —— 能拿 70% 的分数
2. 有时间再优化 —— 空间换时间
3. **408 不抠语法，扣算法思想**。暴力解写出思路就能拿大半分。

### 阶梯难度指引（不同阶段做不同题）

| 阶段 | 做题范围 | 说明 |
|:-----|:---------|:-----|
| **第一幕（抢梁柱）** | 王道课后选择题 + 简单大题 | 先验证概念是否理解，不做综合题 |
| **第二幕（砌砖墙）** | 王道课后题 + 选择题真题 + 典型大题 | 开始接触真题风格 |
| **第三幕（精装修）** | 全部真题套卷 + 错题重做 | 严格计时，模拟考场 |

**做题顺序**：课后选择题 → 课后大题 → 选择题真题 → 大题真题 → 套卷

> **第一幕就做真题大题，80% 做不出来，归因变复杂，严重打击信心。** 题要随着能力升级。|

## 第三章：纠错方法论（错题是唯一的老师）

### 拿到错题，先定级

| 错误类型 | 特征 | 处理方式 |
|:---------|:-----|:---------|
| **A 级（核心错）** | 概念混淆、思维偏差 | **进闭环**：深度归因 + 补充笔记 |
| **B 级（偏难怪）** | 冷门考点、超纲题 | **打标记**：书上标红，不进错题本 |
| **C 级（粗心错）** | 计算错、审题错 | **罚抄**：把数字/单位抄 3 遍，放过自己 |

### A 级错题的四阶归因

每道 A 级错题，问自己 4 个问题：

1. **归类**：这是知识遗忘？概念混淆？思维偏差？还是没见过这类题型？
2. **溯源**：这个知识点属于哪一章？和哪个概念容易搞混？
3. **解决**：针对这个错因，我具体做什么？（画对比表？补充模型？总结范式？）
4. **熔断**：这个知识点我已经补了 3 次还是错？→ **打入冷宫**，下月解冻再看

### 涂料级自动升舱（真题驱动的预警）

涂料级虽然不进闭环，但**真题是唯一的风向标**。如果真题反复考到同一个冷门，它就不再是冷门。

```
做历年真题选择题时，碰到涂料级的知识点：

① 错第 1 次 → 在书上打个红圈 ⭕
② 同一个涂料级知识点，在真题中错够 2 次
   → 触发自动升舱：从"涂料"晋升为"砖墙级"
   → 在下周安排 3 道相关选择题补练
   → 该知识点从配装表上永久移除"涂料"标签
```

### 冷宫 → 低温休眠（每月唤醒）

被打入冷宫的知识点不能彻底不管。每月唤醒两次，只混眼熟：

```
每月 1 号、15 号 → 冷宫解冻日

执行 10 分钟：
① 打开冷宫列表
② 每个知识点只看真题标准答案的"第一步"和"最后结果"
③ 不深究原理，只混个眼熟
④ 10 分钟到，立即停止
```

### 错题归档格式（越简单越好）

```
7/13 | 计组 | Cache 组号 vs 标记位分不清
→ 画了地址拆分图贴墙上
→ 7/20 翻一眼
```

**关键：错题本不用写得太美，要写得"能找回"。**

## 第四章：时间调度方法论（一周怎么过）

### 三幕式战略

| 阶段 | 时间 | 核心任务 | 质量门标准 |
|:-----|:-----|:---------|:-----------|
| **抢梁柱** | 现在 – 8 月底 | 只学高频章节（梁柱），低频直接跳过 | 正确率 60%（允许带病前进） |
| **砌砖墙** | 9 月 – 10 月底 | 补中频章节（砖墙）+ 跨科缝合 | 正确率 70%（严格执行） |
| **精装修** | 11 月 – 考前 | 刷套卷 + 错题闭环 | 追踪套卷得分率 |

### 缝合测试具体执行方式（跨科联动专用）

跨科大题（每年至少 1 道，约 15 分）是最容易丢分也最容易提分的板块。**不是靠"多学"，是靠"专门练"。**

```
执行步骤：

① 找一道标注为"OS+计组"联考的综合题真题
② 只读题干，不看答案
③ 画一条"数据流路径图"：
   数据从磁盘出发，经过 OS 页表转换，
   进入内存，经过计组 Cache 映射，
   最终到达 CPU——每一步发生了什么？

④ 打开标准答案，对照修正你的路径图
⑤ 把最终的路径图贴在笔记上，旁边写：
   "缝合点：计组 Cache ↔ OS 页表"
```

**什么时候做？** 学完 Cache（计组）+ 学完页表（OS）后，主动找一道混合真题做一次。

### 每日节奏——三档模式（根据身体状态切换）

**不要问"我该用什么模式"，先看身体信号：**

| 身体信号 | 切到什么模式 | 做什么 |
|:---------|:-------------|:-------|
| 身体充满电（不困不饿） | **🔥 满血模式** | 只做理解型任务：学新知识、做大题、画图谱。死磕 20 分钟不翻答案 |
| 身体有点沉（刚吃完饭/微困） | **🚙 巡航模式** | 只做执行型任务：刷选择题、整理错题、默写话术。可以翻答案 |
| 身体极度抗拒（眼皮打架） | **🛵 苟活模式** | 只做物理型任务：抄答案、抄定义、整理书桌。最多 15 分钟，到点合书 |

> **状态差的时候做"深度归因"= 酒后驾车，必出事故。**
> **状态差的时候，只做"物理抄写"。**

### 15 分钟热试协议（每天的第一个动作）

**无论今天打不打算学习，坐下来先做这件事：**

```
① 翻开书，抄题干 / 抄公式 / 抄目录
② 只做"物理抄写"，不动脑
③ 15 分钟后，身体会告诉你答案：
   - 越抄越精神 → 无缝切换到【满血模式】
   - 越抄越困 → 切换到【苟活模式】，15 分钟到点立刻合书睡觉
```

> **原理：用"身体反馈"代替"大脑决策"。你不需要判断自己什么状态，抄 15 分钟，身体会替你判断。**

### 每日时间参考

| 时段 | 自然状态 | 推荐模式 | 适合任务 |
|:-----|:---------|:---------|:---------|
| 08:00-08:15 | 刚醒，还没进入状态 | 15 分钟热试 | 抄定义/抄题干 |
| 08:15-11:30 | 精力最充沛 | **🔥 满血模式** | 学新知识、做大题、画图谱 |
| 14:00-17:00 | 午后容易困 | **🚙 巡航模式** | 刷选择题、整理错题、默写模板 |
| 19:00-21:00 | 恢复一些精力 | 灵活切换 | 查漏补缺、跨科缝合、看答案倒推 |
| 22:30 | 该停机了 | **🛵 苟活模式** | 如果还想学，只抄答案，不学新东西 |

### 日模式 × 月幕次的联动规则

三档模式（满血/巡航/苟活）由 **身体状态** 决定，三幕式（抢梁柱/砌砖墙/精装修）由 **当前进度** 决定。两者互不冲突，组合规则如下：

```
日模式决定"做什么类型的任务"，月幕次决定"任务的内容范围"。

第一幕（抢梁柱）：
├── + 满血模式 = 学梁柱章节的新知识
├── + 巡航模式 = 刷梁柱章节的选择题 + 课后题
└── + 苟活模式 = 抄梁柱章节的标准答案

第二幕（砌砖墙）：
├── + 满血模式 = 学砖墙章节 + 做缝合测试
├── + 巡航模式 = 刷砖墙章节选择题 + 真题选择题
└── + 苟活模式 = 整理错题 + 冷宫解冻

第三幕（精装修）：
├── + 满血模式 = 做套卷（严格计时，模拟考场）
├── + 巡航模式 = 重做错题 + 整理笔记
└── + 苟活模式 = 默写解题范式 + 抄答案

一句话：状态决定怎么学（满血/巡航/苟活），进度决定学什么（梁柱/砖墙/套卷）。
```

### 每周日的固定动作

1. **找瓶颈（约束理论）**：问自己"过去一周，限制我总分上涨的最大科目/章节是什么？"→ 下周 50% 的时间砸向这里。其他科目哪怕学得再烂也先放一放——因为任何系统在任何时刻只有一个瓶颈。
2. **归零**：把这一周的废草稿纸全部撕掉扔掉，只留错题卡片
3. **不学新东西**：周日不做新题，不补旧账

## 第五章：心态管理的 3 条铁律

### 铁律 1：烂开始

- 不追求"进入状态再学"
- **只要翻开书，写下第一个字，就算今天的学习已经开始**
- 允许自己先瞎写 5 分钟（抄目录、抄题干、抄公式）

### 铁律 2：垃圾时间合法

- 每天允许 2 个"垃圾番茄钟"（发呆、上厕所、刷手机）
- **不算违规，算系统维护**
- 强行不让大脑休息，它会在考场上报复你

### 铁律 3：不断档 > 学得多

- 状态差的时候，只做一件事：**抄 15 分钟答案，然后合书睡觉**
- 不求懂，只求手熟
- 今天能活着学完，就算赢。**系统允许你当废物，但系统不允许你断档。**
