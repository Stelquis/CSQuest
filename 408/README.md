## 🧭 使用方式

### 学习流程

```
章节学习 → 整理笔记(notes) → 做习题(exercises) → 
← 错题归入对应目录 → 阶段总结(summary.md) →
真题实战 → 错题本回顾
```

### 笔记规范（建议）

每章笔记统一格式：

```markdown
# 第 X 章 — 标题

## 核心概念
- ...

## 重点公式 / 算法
- ...

## 易错点
- ...

## 真题链接
- 年份-题号(简要说明)
```

### 习题命名

```
exercises/
├── 课后/
│   ├── ch02-线性表.md
│   └── ...
├── 专项/
│   ├── pv操作.md
│   └── ...
└── 错题/
    └── ...
```
